Web Sudoku

A sudoku game using bootstrap and angular that targets all devices.

The objective of the game is to fill the entire grid so that the digits from 1 to 9 appear exactly once in each region.

Features
  • Jul 15, 2014: User login to Firebase.
  • Jul 14, 2014: Keyboard input in game play.
  • Jul 11, 2014: Validation to complete a game.
  • Jun 24, 2014: Timer and candidate display.
  • Jun 19, 2014: Loading screen for better user experience.
  • Jun 16, 2014: Load puzzles from Firebase.
  • Jun 13, 2014: Grid display Sudoku board for all sizes.
  • Jun 12, 2014: Apple-style UI for settings.
  • Jun 11, 2014: Responsive UI for all devices.
